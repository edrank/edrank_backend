## Setting up locally

- Clone the repo
- Install go from [here](https://go.dev/dl/)
- Check go installation - `go version`
- Move to root directory of the project
- On windows
    - `spin.bat` OR `go run main.go`
- On Linux
    - `bash spin.sh` OR `go run main.go`
